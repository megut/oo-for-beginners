Klassen - Attribute

Eine Klasse kann als Schablone verstanden werden.
Anhand des Beispiels der Klasse "Auto" lässt sich das gut erklären, was genau eine Klasse, beziehungsweise eine Schablone, ist. Jedes Auto, was mit Benzin oder Diesel betankt wird, hat eine Anzahl an Sitzen und Türen, eine Farbe, eine Marke und auch einen Spritverbrauch. Diese „Attribute“ sind also vorgegeben. Jedes Attribut wird einem Datentypen oder einer anderen Klasse zugewiesen.
Auto
anzahlSitze: int
anzahlTüren: int
farbe: String
marke: String
verbrauch: double
