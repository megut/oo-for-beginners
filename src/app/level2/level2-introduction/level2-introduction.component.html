Arrays

Im Gegensatz zu den bereits vorgestellten Datentypen können Arrays mehrere Werte gleichzeitigt speichern. Allerdings können nur Werte des gleichen Datentypen gespeichert werden. Arrays haben eine feste Länge.
{{i}}
Arrays kann man vergleichen mit Güterzügen. Ein Güterzug mit der Länge fünf, hat fünf Güterwagons. Die Nummerierung beginnt bei null. Innerhalb eines Güterwagons kann ein Gegenstand einer Kategorie transportiert werden. Die Kategorie wird für den gesamten Güterzug festgelegt. Es können beispielsweise in einem Güterzug nur Obstsorten transportiert werden. In den einzelnen Wagons kann jeweils eine Frucht liegen.
