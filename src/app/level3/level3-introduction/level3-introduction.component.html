Schleifen

Durch die Verwendung von Schleifen, können Anweisungen so oft wiederholt werden, bis die Abbruchsbedingung(en) erfüllt werden.

Ohne Schleife

Zum ersten Auto gehen

Prakticket prüfen

Zum nächsten Auto gehen

Prakticket prüfen

Zum nächsten Auto gehen

Parkticket prüfen

...

Mit Schleife

Zum ersten Auto gehen

Solange weitere Autos vorhanden sind

Parkticket prüfen

Zum nächsten Auto gehen
